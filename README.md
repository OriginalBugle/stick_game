## Installation
1.Clone the repository:
   ```shell
   git clone https://github.com/OriginalBugle/stick_game.git
   ```
2.Navigate to the project directory:
```shell
cd stick_game
```


  3. Install the required dependencies:
  ```shell
  pip install requirements.txt
  ```
